# `pair_ns_tests/` — `Pair` namespace tests

Tests for the `Pair` "class" defined in `../concept_checks/pair_ns.fj` (a
namespace used as an object with two number fields and methods). Each test is
compiled together with `pair_ns.fj` and checks one group of methods.

| File | Purpose |
|---|---|
| `test1.fj` | `init` and `print` (expects two zeroed pairs). |
| `test2.fj` | `add`, `add_first` and `add_second` across two pairs. |
| `test3.fj` | `swap`. |
